# Theme Customizer

FXT theme provides extensive customization options through the WordPress Customizer, allowing you to modify colors, styles, and appearance without editing code.

---

## Accessing the Customizer

To access the theme customizer:

1. In WordPress dashboard, go to **Appearance** â†’ **Customize**
2. The customizer will open with a live preview of your website
3. Use the left sidebar to navigate through different sections

---

## FXT Styles Panel

The **FXT Styles** panel contains all theme-specific customization options.

### Theme Colors

Customize the main color scheme of your website.

**Available Colors**:

- Body Text Color: Main text color throughout the site
- Background Color: Main background color
- Button Color: Primary button background color
- Review Links Color: Color for review and rating links

### Header Colors

Customize header-specific colors and styling.

**Available Colors**:

- Header Background Color: Main header background
- Header Text Color: Header text and navigation color
- Header Link Color: Header navigation link color
- Header Link Hover Color: Header link hover state

### Footer Colors

Customize footer-specific colors and styling.

**Available Colors**:

- Footer Background Color: Main footer background
- Footer Text Color: Footer text color
- Footer Link Color: Footer link color
- Footer Link Hover Color: Footer link hover state

---

## Live Preview

All changes made in the customizer are displayed in real-time in the preview panel:

1. Make changes in the left sidebar
2. See immediate updates in the preview
3. Changes are not published until you click **Publish**
