# FXT Integrations

## tablrr Integration

[tablrr.app](https://tablrr.app?utm_source=dinomatic&utm_medium=docs&utm_term=fxt) is a centralized listings platform built for forex and finance affiliates managing multiple websites. Instead of updating broker data on each site individually, you manage everything from one dashboard — and all your connected websites stay in sync automatically.

With the FXT integration it goes even further: **create your brokers in tablrr and have their fields sync directly into WordPress broker reviews across all your websites**.

Connect your FXT-powered WordPress websites to tablrr and your broker review pages stay in sync automatically. Any broker data you update in tablrr — trading conditions, ratings, regulations, pros and cons — flows back to your websites without any manual work.

### Requirements

- FXT theme version 1.5 or higher
- tablrr account (Hobby or higher)
- [tablrr WordPress plugin](https://tablrr.app/wordpress-plugin?utm_source=dinomatic&utm_medium=docs&utm_term=fxt) installed on your site

### How to Connect

1. In the [tablrr dashboard](https://app.tablrr.app?utm_source=dinomatic&utm_medium=docs&utm_term=fxt), go to **Settings → Sites** and click **Add Site**
2. Enter your domain and click **Add** — you'll receive a verification code
3. In your WordPress admin, go to the tablrr plugin settings and paste the verification code
4. The plugin verifies your domain automatically and creates an API token

That's it. Your site is connected. For more detail on site verification, see [tablrr sites guide](https://docs.tablrr.app/sites?utm_source=dinomatic&utm_medium=docs&utm_term=fxt).

### How to Enable Broker Data Sync

On the same page, enable **Operator Sync** checkbox in the plugin settings.

After that there will be an "operator mapping" section available. This allows you to map operators (forex brokers) in your tablrr account to broker review pages in your website. You need to set it up once and it will work without any issues afterwards.

And you're done, you broker data will begin syncing automatically.

### What Syncs

Every time the sync runs, the plugin fetches the latest broker data to your FXT review pages:

- Name, overall rating, description, affiliate link, and terms
- Trading conditions — minimum deposit, spreads, leverage, instrument counts, pair types
- Individual ratings (up to six value)
- Pros and cons
- Deposit and withdrawal methods
- Trading platforms and regulatory bodies
- Restricted countries and supported languages
- Account features — demo account, mobile trading, swap-free, copy trading, and more

### Sync Schedule

The plugin checks for updates every hour. Any broker record changed in tablrr since the last check is pulled in and applied to the corresponding WordPress post automatically.
