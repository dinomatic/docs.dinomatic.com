# Integrations

Kemoku can be extended with third-party services to automate your workflow and manage data more efficiently.

## tablrr - Automated Data Management

Connect your Kemoku-powered sites to [tablrr.app](https://tablrr.app?utm_source=dinomatic&utm_medium=docs&utm_term=kemoku) to manage your operator data, bonus details, and affiliate links from a single dashboard.

- **Automatic Sync**: Update data in one place and see changes across all your sites.
- **Centralized Control**: Manage multiple Kemoku installations without logging into each WordPress admin.

Tutorial: [Learn how to set up the Kemoku-tablrr integration](https://dinomatic.com/posts/how-to-sync-kemoku-review-fields-across-multiple-websites-using-tablrr).
