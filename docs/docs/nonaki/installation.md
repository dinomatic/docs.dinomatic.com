# Plugin Installation

You can install the plugin either through the WordPress dashboard (recommended) or via FTP.

---

## Before Installation

Make sure your server meets the minimum [system requirements](/docs/nonaki/) and download the plugin file from your [DinoMatic account](https://dinomatic.com/account).

### Install through WordPress Dashboard

1. Visit WordPress dashboard → **Plugins** and click `Add New`, then `Upload Plugin`
2. Click `Choose File`, select `nonaki.zip` and click `Install Now`
3. Activate the plugin

### Install via FTP

1. Log in to your server using an FTP client, cPanel, or similar
2. Navigate to the `plugins` directory in your WordPress root
3. Upload `nonaki.zip` and unzip it — you should have a `nonaki` directory in `plugins`
4. Remove the `nonaki.zip` file
5. Visit WordPress dashboard → **Plugins** and activate the plugin
